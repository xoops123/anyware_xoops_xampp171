ABOUT

どこでもXOOPSとは
Windows で、web applicationを体験できるパッケージであるXAMPPをベースとして、XOOPS Cube Legacy 2.2（pack2011rc）をインストールしたものです。インストールの手間を省くとともに、主要な設定やサンプルデータが入力済みなので、手軽に XOOPS を体験することができます。
（ログインID・パスワード： admin ）

XCL2.2とは
最新のXOOPS Cube Legacy 2.2 の略称です。

Pack2011とは
2011年に自然発生的な技術者集団による草の根活動により構築された新しいディストリビューションです。最新のXCL2.2をベースにしているのが特徴です。また、投票により選出されたモジュールを組み合わせ、それぞれが継続メンテナンスされています。

★ ご利用の前に、XAMPP Control Panel にて、Apache と MySQL を起動してください。
